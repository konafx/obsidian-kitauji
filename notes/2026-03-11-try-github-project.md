Roadmap, BoardともにOKっぽそう

Estimatedなどはcustom fieldで設定する
## View
### Roadmap
- Group By: Parent issue
- Markers: Iteration
- Dates: Iteration
- Field sum: Estimated

### Board
- Filter: iteration:@current type:Task
- Column by: Status
- Swimlane: Parent issue
- Field sum: Count, Estimated
- Slice by: Assignees

## Issue
- Typeを設定する。Feature, Task, Bugが設定可能
- sub issueが作れる。便利
	- `- [ ] task`記法から勝手にsub issue作ってくれたら最高

## 気になる点
- FeatureにEstimatedなどをいれても Ignore Featureとかしないとなんかなー
- あとFeature, Taskであまり見た目の区別がないのも気になる
